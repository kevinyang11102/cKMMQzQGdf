# 健身房管理系统 java149

## 项目概述

本项目是基于SpringBoot开发的健身房管理系统，整合了前后端代码（采用JSP技术），为健身房提供了一个集会员管理、教练管理、课程管理、器材管理和商品管理于一体的解决方案。

## 技术栈

- **开发语言**：Java
- **开发框架**：SpringBoot
- **前端技术**：HTML, JSP
- **数据库**：Mysql

## 系统角色

系统主要分为以下角色：

- 会员
- 教练
- 管理员

## 运行环境

- **系统环境**：Windows，Mac
- **开发环境**：IDEA、JDK1.8、Mysql

## 功能模块

### 会员管理

- 会员列表
- 会员到期提醒
- 会员充值管理
- 会员卡续费
- 会员卡缴费记录
- 会员卡类型管理

### 教练管理

- 教练列表
- 会员私教课程列表
- 会员私教详情

### 课程管理

- 课程列表
- 课程预约

### 器材管理

- 器材列表
- 器材维护

### 物品管理

- 物品遗失登记
- 物品归还处理

### 商品管理

- 商品分类
- 商品销售情况统计

### 信息统计

- ECharts数据可视化

## 部署步骤

1. 准备运行环境：确保IDEA、JDK1.8、Mysql已安装
2. 克隆或下载项目代码
3. 导入数据库文件
4. 使用IDEA打开项目，并配置数据库连接
5. 运行项目

## 核心亮点

- **前后端整合**：项目采用前后端不分离模式，便于部署和维护。
- **功能全面**：覆盖健身房业务主要流程，包括会员、教练、课程、器材和商品等管理。
- **数据可视化**：通过ECharts进行数据统计与展示，使管理更加直观。
- **灵活配置**：可根据不同健身房需求进行功能配置和调整。
